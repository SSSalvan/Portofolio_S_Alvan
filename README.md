This is my personal portfolio website highlighting my technical abilities, completed projects, and professional achievements.

## Features

- Project showcase

- About section

- Smooth animations and interactions
- Modern UI/UX
- Mobile optimized

## Tech Stack

HTML5
CSS3
JavaScript
Responsive Web Design

## Installation

```bash
git clone https://github.com/SSSalvan/Portfolio-SalvanS.git
cd Portfolio-SalvanS
```

No build process required. Simply open index.html in a web browser.

## Usage

For local development:

```bash
python -m http.server 8000
```

Or use any local server. Then open http://localhost:8000
