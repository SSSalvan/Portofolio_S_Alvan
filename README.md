# Personal Portfolio Website

## Overview
The source code for my professional portfolio website. It showcases my technical projects, research experience, and engineering capabilities across Machine Learning, Computer Vision, and Software Development.

## Technical Architecture
- **Frontend:** HTML5, CSS3, JavaScript
- **Deployment:** GitHub Pages / Vercel

## Features
- Modern, responsive design suitable for mobile and desktop screens.
- Project galleries highlighting key technical achievements.
- Contact integration and resume download capabilities.
